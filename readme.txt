LiteClass PWA
-------------
Files:
  - index.html
  - style.css
  - app.js
  - manifest.json
  - sw.js
  - icons/icon-192.png
  - icons/icon-512.png

How to use:
  1. Host these files (GitHub Pages or open index.html locally).
  2. Visit the app online once (to cache CDN/JS) or add jszip.min.js locally.
  3. Teacher: create class, upload audio & slides, enter timings (mm:ss), click Build class ZIP.
  4. Student: upload class ZIP and press Start. Use Start/Pause/Reset or 10s back/forward.
  5. Admin password: liteclass (local subject list stored in localStorage).
